# Personal Expense Tracker

A simple **Python-based expense tracker** that allows you to record, view, and analyze your personal expenses. This project uses **CSV files** to store data and **pandas** for basic analysis like category-wise summaries and total spending.

---

## Features

- **Add Expenses:** Add your daily expenses with amount, category, and notes.  
- **View Expenses:** See all recorded expenses in a tabular format.  
- **Total Spending Calculation:** Quickly calculate the total money spent.  
- **Category-wise Summary:** See how much you spend in each category.  
- **Set Budget:** Set a monthly or daily budget to track your spending.  
- **Check Budget:** Compare total expenses against your budget.  
- **Reset Data:** Clear all recorded expenses safely.  

---

## Technologies Used

- **Python 3.x**  
- **CSV files** for data storage  
- **pandas** library for data manipulation and analysis  

---

## How to Use

1. **Clone the repository**
```bash 
    git clone https://github.com/your-username/personal-expense-tracker.git
```
2.**Navigate to the project directory**
```bash 
    cd personal-expense-tracker
```
3.**Install dependencies (if not installed)**
```bash 
    pip install pandas
```
4.**Run the program**
```bash 
    python main.py
```
